# ⚡ Energy Consumption Analysis Dashboard

## 📌 Project Overview

This project analyzes energy consumption patterns using Power BI to identify peak demand periods, temperature impacts, and time-based usage trends.

The objective was to transform raw energy data into actionable insights that can support energy planning and demand management.

---

## 🛠 Tools & Technologies

- Power BI
- DAX
- Power Query
- Excel / CSV
- Data Visualization
- Feature Engineering

---

## 📊 Dashboard Pages

### 1. Executive Summary
- Total Energy Consumed
- Average Power Usage
- Maximum Power Recorded
- Peak Events Count
- Annual Energy Consumption Trend

### 2. Peak Consumption Analysis
- Temperature vs Peak Power Analysis
- Peak Event Distribution
- Peak Power by Temperature Category

### 3. Energy Usage Patterns & Time Analysis
- Day vs Time Block Heatmap
- Energy Consumption by Time Block
- Weekday vs Weekend Usage Analysis

---

## 🔍 Key Insights

- Energy consumption peaked during evening hours.
- Higher temperatures were associated with increased peak power demand.
- Peak events represented approximately 11% of observations.
- Weekday energy consumption exceeded weekend consumption.
- Overall energy usage showed a declining trend over the observed period.
